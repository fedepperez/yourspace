YourSpace - Un clone di MySpace

Un tuffo nel passato con tecnologia moderna. Un social network completo con autenticazione sicura, profili personalizzabili e temi dinamici.

Cosa c'è dentro?

Auth System

Registrazione e Login completi con hashing delle password (Bcrypt) e gestione sicura delle sessioni PHP.

Profili & Temi

Ogni utente può modificare la bio e scegliere un tema personalizzato che viene salvato nel database.

Tech Stack

PHP Nativo
MySQL
HTML5
CSS3 Moderno

Quick Start

$ git clone https://github.com/fedepperez/yourspace
# 1. Scarica il progetto in htdocs
$ mysql -u root -p
# 2. Crea DB 'myspace_clone' e importa script.sql
$ code config.php
# 3. Configura le credenziali del DB
Server avviato su localhost!

Requisiti

  • XAMPP, MAMP o LAMP Stack
  • PHP 7.4 o superiore
  • Database MySQL locale
